Automated Justification

Enter a profile of preferences and choose an outcome. Then ask the computer to find a convincing justification for your choice.

    About

    This demo illustrates our research on computing and explaining normatively grounded justifications for collective decisions. Read the demo description to find out more. For an easy-going introduction to the topic, watch this video.

    For in-depth information about our research, consult these papers:

    • Boixel, Endriss. Automated Justification of Collective Decisions via Constraint Solving. AAMAS-2020.
    • Boixel, De Haan. On the Complexity of Finding Justifications for Collective Decisions. AAAI-2021.
    • Boixel, Endriss, De Haan. A Calculus for Computing Structured Justifications for Election Outcomes. AAAI-2022.
    • Nardi, Boixel, Endriss. A Graph-Based Algorithm for the Automated Justification of Collective Decisions. AAMAS-2022.

    This is an open-source project. You will find our code at this GitHub repository.
